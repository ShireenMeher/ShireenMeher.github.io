# Shireen Meher Chirravuri — Portfolio

**Live:** [shireenmeher.github.io](https://shireenmeher.github.io)

Personal portfolio site built with vanilla HTML, CSS, and JavaScript. No frameworks, no build step.

## Features

- Animated particle network background
- Custom cursor with color trail
- Typewriter hero text
- Scroll-reveal animations
- Sections: Skills, Passions, Work Experience, Projects, Education, Contact

## Run locally

Just open `index.html` in any browser — no server needed.
